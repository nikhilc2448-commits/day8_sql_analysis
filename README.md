# Day 8 - SQL for Data Analysis

## Objective

This project demonstrates both fundamental and advanced SQL concepts by analyzing retail business data using the Customers and Orders datasets. The assignment covers database creation, data retrieval, business reporting, joins, subqueries, window functions, CASE statements, and Common Table Expressions (CTEs).

## Dataset

The project uses the following datasets:

- Customers
- Orders

## Database

MySQL 8.0

## Topics Covered

- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- LIMIT
- AND, OR, IN, BETWEEN, LIKE
- Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Subqueries
- CASE WHEN
- Window Functions (ROW_NUMBER, RANK)
- Common Table Expressions (CTE)

## Business Questions Solved

### Basic SQL

1. Display all customer details.
2. Find customers from a specific region.
3. List the top 5 highest-value orders.
4. Calculate total sales by category.
5. Find customers with purchases above a given threshold.
6. Display monthly sales.
7. Categorize orders using CASE statements.
8. Retrieve orders with sales above the average.
9. Display customer and order details using JOINs.
10. Generate business insights using aggregate functions.

### Advanced SQL

1. Find the top 5 customers by total sales.
2. Find customers who placed more than 5 orders.
3. Find the region with the highest average order profit.
4. List orders with sales greater than the overall average sales.
5. Categorize orders as High Value, Medium Value, or Low Value.
6. Find the most popular category in each region.
7. Identify customers who have never placed an order.
8. Calculate total profit for each customer and rank them.
9. Find the month with the highest total sales.
10. Use a Common Table Expression (CTE) to calculate customer profit and filter customers above a profit threshold.

## How to Run

1. Create a MySQL database.
2. Import the Customers and Orders datasets.
3. Execute the SQL queries in order.
4. Review the output for each business question.

## Key Insights

- Identified the top-performing customers based on total sales.
- Compared customer purchasing behavior using order counts.
- Determined the region with the highest average profit.
- Categorized orders into different value groups using CASE statements.
- Ranked customers based on total profit using window functions.
- Identified the highest-performing sales month.
- Used subqueries and CTEs to simplify complex business reporting queries.
